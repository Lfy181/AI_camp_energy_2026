# 能源价格预测与电池充放电策略优化

本项目基于能源系统边界条件（系统负荷、风光发电、联络线等）预测节点电价，并生成最优的电池充放电策略，实现收益最大化。

## 项目概述

- **电价预测**: 使用多种机器学习模型预测节点电价 'A'
- **策略优化**: 根据预测电价，通过优化算法生成最优的电池充放电策略
- **充放电规则**:
  - 充电: 8个时间点（2小时），功率 -1000MW
  - 放电: 8个时间点（2小时），功率 +1000MW
  - 目标: 最大化收益 = 放电收入 - 充电成本

## 关键发现

### 1. 预测准确率 ≠ 经济收益 ⭐
- **最高RMSE (6.1) 的模型收益最好**！
- 关键在于准确预测"什么时候价格低/高"，而非精确预测价格值
- MLP可能捕捉到了电价的相对高低趋势

### 2. 简化增强特征的效果 ⭐
- **周期编码** (hour_sin/cos, dayofweek_sin/cos) 帮助MLP理解时间模式
- **时段标记** (is_morning_peak, is_evening_peak) 区分高峰低谷
- **交互特征** (负荷×风光等) 捕捉非线性关系

### 3. 最佳方案推荐

**🥇 最高收益方案**: MLP + 简化增强特征 + 暴力搜索 ⭐
- 总收益: 3,850,900.64元
- 提升: +453.01%
- RMSE: 6.10
- 特点: 收益爆炸性增长！

**🥈 次优方案**: MLP + 基础特征 + 暴力搜索
- 总收益: 837,523.25元 (+20.27%)
- 特点: 简单、快速、稳定

## 简化增强特征详解 (35个特征)

### 基础特征 (7个)
- 系统负荷预测值、风光总加预测值、联络线预测值
- 风电预测值、光伏预测值、水电预测值、非市场化机组预测值

### 时间特征 (7个)
- hour, minute, dayofweek, month
- is_weekend, quarter, dayofyear

### 时段标记 (4个)
- is_morning_peak (7-11点)
- is_evening_peak (17-21点)
- is_night (0-6点)
- is_afternoon (12-17点)

### 周期编码 (8个)
- hour_sin, hour_cos
- dayofweek_sin, dayofweek_cos
- month_sin, month_cos
- dayofyear_sin, dayofyear_cos

### 交互特征 (9个)
- 乘法: 负荷×风光, 负荷×风电, 负荷×光伏, 风光×风电, 风光×光伏, 风电×光伏
- 除法: 负荷/风光, 风光/风电, 风光/光伏

## 核心洞察

### 为什么MLP在增强特征下表现这么好？

1. **时间模式学习**: 周期编码让MLP理解电价每天都有的周期性变化
2. **非线性建模**: MLP自动学习特征间复杂交互
3. **相对预测**: 不追求精确值，而是准确预测价格相对高低
4. **价差最大化**: 找到最大的充放电价差时机

### 关键启示

1. **简单模型 + 好的特征 = 好效果**
2. **RMSE不是唯一指标**: 经济收益才是最终目标
3. **特征工程 > 模型调参**: 好的特征比复杂模型更重要
4. **暴力搜索的威力**: 在小规模问题上，暴力搜索能找到全局最优
