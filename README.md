# 能源价格预测与电池充放电策略优化

本项目基于能源系统边界条件（系统负荷、风光发电、联络线等）预测节点电价，并生成最优的电池充放电策略，实现收益最大化。

## 项目概述

- **电价预测**: 使用多种机器学习模型预测节点电价 'A'
- **策略优化**: 根据预测电价，通过优化算法生成最优的电池充放电策略
- **充放电规则**:
  - 充电: 8个时间点（2小时），功率 -1000MW
  - 放电: 8个时间点（2小时），功率 +1000MW
  - 目标: 最大化收益 = 放电收入 - 充电成本

## 消融实验设计

### 实验配置

| 配置维度 | 选项 | 说明 |
|---------|------|------|
| **机器学习模型** | GradientBoost, XGBoost, LightGBM, RandomForest, Ridge, MLP | 多种主流回归模型对比 |
| **特征工程** | 基础特征 / 增强特征 | 11个特征 vs 700+个特征 |
| **超参数优化** | 无 / Optuna贝叶斯优化 | 5次试验的自动调参 |
| **策略算法** | 暴力搜索 / DP优化 | O(n²) vs O(n)复杂度 |

### 实验分组

**第一阶段**: 基础特征实验 (Exp0-Exp5)
- 验证不同模型在基础特征下的表现
- 探索模型选择对收益的影响

**第二阶段**: 增强特征实验 (Exp6-Exp12)
- 验证增强特征工程的效果
- 完整的模型+特征+优化组合

## 关键发现

### 1. 预测准确率 ≠ 经济收益
- 最低RMSE (0.5564) 的模型不一定带来最高收益
- 简单的线性模型(Ridge)和神经网络(MLP)反而表现更好

### 2. 特征工程至关重要
- 增强特征工程平均降低RMSE约13%
- 滞后特征、滚动统计、交互特征显著提升预测质量

### 3. 最佳方案推荐

**🥇 最高收益方案**: MLP (Exp5)
- 总收益: 837,523.25元 (+20.27%)
- 特点: 简单、快速、对基础特征表现优异

**🥈 综合最优方案**: XGBoost完整优化 (Exp11)
- 总收益: 784,856.94元 (+12.7%)
- RMSE: 0.5564 (最低)
- 特点: 预测最准确，计算效率高

## 快速开始

### 1. 安装依赖

```bash
pip install pandas numpy scikit-learn xgboost lightgbm optuna
```

### 2. 下载数据

```bash
pip install modelscope
modelscope download --dataset Datawhale/AI_camp_energy_2026
```

### 3. 运行实验

```bash
# 运行完整消融实验（13组）
python run.py

# 或只运行增强特征实验
python run_remaining_experiments.py
```

### 4. 查看结果

实验结果保存在 `output/experiments/` 目录：
- `experiment_results_*.csv` - 所有实验汇总指标
- `Exp*_output.csv` - 各实验的充放电策略

## 特征工程详解

### 基础特征 (11个)
- 7个系统边界特征: 系统负荷、风光总加、联络线、风电、光伏、水电、非市场化机组
- 4个时间特征: hour, minute, dayofweek, month

### 增强特征 (700+个)
1. **时间特征增强**
   - 周期编码: hour_sin/cos, dayofweek_sin/cos, month_sin/cos
   - 时段标记: is_morning_peak, is_evening_peak, is_night, is_afternoon
   - 其他: is_weekend, quarter, dayofyear, weekofyear

2. **滞后特征**
   - 短周期: 1, 2, 3, 4, 8, 16, 24 (分钟级别)
   - 长周期: 48, 96, 192, 672 (小时到周级别)

3. **滚动统计特征**
   - 窗口大小: 4, 8, 16, 24, 48, 96, 192
   - 统计量: mean, std, min, max, median

4. **变化率特征**
   - 差分: diff_1, diff_4, diff_8
   - 百分比变化: pct_change_1, pct_change_4

5. **聚合特征**
   - 按小时聚合: mean, std
   - 按星期聚合: mean, std
   - 偏离度: dev_from_hour_mean, dev_from_dow_mean

6. **交互特征**
   - 特征乘法: 系统负荷×风光, 风电×光伏 等
   - 特征除法: 系统负荷/风光, 风光/光伏 等

## 后续优化方向

1. **集成学习**: 融合多个模型的预测（Stacking/Voting）
2. **特征选择**: 基于特征重要性的特征筛选
3. **更多HPO试验**: 从5次提升到50次
4. **时序模型**: 引入LSTM、Transformer等深度学习模型
5. **市场因素**: 添加供需关系、历史电价等外部特征
6. **动态策略**: 考虑实时调整的多阶段优化
